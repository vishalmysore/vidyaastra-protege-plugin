Vidyaastra

A Protégé Desktop plugin for AI Integration with Large Language Models and Interactive Graph Visualization. The plugin provides two main features:

  1. AI Integration - Available in menu "Tools > AI Integration - Send Query to LLM..."
  2. VidyaAstra Graph - A tab view showing interactive ontology graph visualization

Features

AI Integration

  • Interactive AI query interface integrated into Protégé
  • Context-aware ontology information display
  • Clean and intuitive user interface for query input and response display
  • Easy-to-extend architecture for LLM API integration

VidyaAstra Graph

  • Interactive graph visualization of ontology structure
  • Multiple layout algorithms (FR, Circle, KK, Spring, ISOM)
  • Visual distinction between classes and object properties
  • Zoom and pan controls
  • Click to select entities
  • Automatic display of subclass relationships

Usage

AI Integration

  1. Open your ontology in Protégé
  2. Navigate to Tools > AI Integration - Send Query to LLM...
  3. Enter your query in the text area
  4. Click "Send Query" to interact with the AI service
  5. View the response in the response panel

Graph Visualization

  1. Open your ontology in Protégé
  2. Go to Window > Views > Ontology views > VidyaAstra Graph
  3. The graph will automatically display classes and their relationships
  4. Use toolbar buttons to switch between different layout algorithms
  5. Use mouse to pan (drag) and zoom (wheel)
  6. Click on nodes to select entities
  7. Click "Refresh Graph" to reload after ontology changes
